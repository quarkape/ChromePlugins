# ChromePlugins
自己制作的谷歌浏览器插件

## browser.crx
- 谷歌浏览器new tab的替代性页面，可以自定义新标签页的标题、favicon以及背景，可以有效解决谷歌浏览器如果设置新标签页非google.com会在页面中间出现曾经打开过的网页的问题。
- 需要后台一直开启，可能会增加少量的功耗
- *使用方法*：直接安装，然后开启即可。如果浏览器任何提示，直接同意或者通过即可
